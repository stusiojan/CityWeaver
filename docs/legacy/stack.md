# Stack

W tym pliku jest lista technologii, które wykorzystuję w projekcie

## Technologie
- Swift packages for backend
- SwiftUI for frontend
- xcodegen for managing xcodeproj from project.yml file
- xcode-build-server for bridging xcodeproj dependencies to LSP (important for development outside of xcode)
- xcbeautify for more readable outputs
- swiftformat for advance code formatting
